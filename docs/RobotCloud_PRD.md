**目标用户**：机器人研发团队 / AI 开发者 / 教育与科研机构

------------------------------------------------------------------------

## 一、项目概述

### 1.1 产品愿景

RobotCloud 致力于打造一个一站式的
**机器人感知与智能开发云平台**，帮助用户从数据采集 → 数据管理 → 模型训练
→ 云端推理 → 硬件部署的完整闭环，加速智能机器人算法研发与验证过程。

### 1.2 核心价值

-   降低机器人感知算法的开发门槛\
-   提供统一的多模态数据管理与可视化能力\
-   支持云端自动化模型训练与仿真部署\
-   为不同层次用户提供渐进式服务体系（Free / Plus / Pro）

## 四、功能需求细化

### 4.1 用户系统

  ---------------------------------------------------------------------------------------
  模块                 功能点                             说明
  -------------------- ---------------------------------- -------------------------------
  手机注册             输入手机号 → 获取验证码 → 设置密码 支持国内手机号验证

  登录                 手机号+密码登录 / 验证码登录       token鉴权（JWT）

  个人中心             查看套餐等级、到期时间、使用记录   支持升级套餐（人工或自助）

  权限管理             free / plus / pro                  影响可访问的页面和后端API权限
  ---------------------------------------------------------------------------------------

#### 权限定义表

  模块                  Free   Plus   Pro
  --------------------- ------ ------ -----
  可视化数据集          ✅     ✅     ✅
  上传数据集            ❌     ✅     ✅
  模型云端训练          ❌     ✅     ✅
  模型云端推理          ❌     ✅     ✅
  模型下载              ❌     ✅     ✅
  定制硬件平台          ❌     ❌     ✅
  仿真训练              ❌     ❌     ✅
  团队协作 / 成员管理   ❌     ❌     ✅

------------------------------------------------------------------------

### 4.2 数据集管理模块

  功能项       描述
  ------------ -----------------------------------------------------
  上传数据集   支持 ZIP / TAR / JSON / MP4 / 点云等格式上传
  数据解析     自动识别数据结构（图像、LiDAR、语义标签等）
  数据预览     可视化展示样本（缩略图、3D点云、视频）
  数据统计     自动生成统计报告（数量、类型、质量检测）
  权限控制     仅 plus/pro 用户可上传，free 用户仅可浏览公开数据集

------------------------------------------------------------------------

### 4.3 模型训练与推理模块

  功能项         描述
  -------------- ---------------------------------------------------------
  模型选择       提供预设模板（YOLO、OccupancyNet、PointTransformer 等）
  参数配置       用户可自定义学习率、batch size、epoch等
  训练任务调度   后端任务队列（Celery / Ray）分发GPU任务
  日志监控       实时可视化 loss、accuracy 曲线
  模型推理       支持对上传数据集进行推理，展示结果（2D框、3D点云检测）
  模型下载       训练完成后可下载 `.pt` 或 `.onnx` 文件
  资源限额       Plus：单任务最大 2 GPU，Pro：最大 8 GPU + 并行任务

------------------------------------------------------------------------

### 4.4 仿真与硬件支持（Pro）

  功能项         描述
  -------------- --------------------------------------------------
  仿真环境       支持 IsaacSim / Gazebo Web 仿真
  场景导入       可上传自定义 3D 环境（.USD / .SDF / .GLB）
  模拟训练       支持强化学习、导航算法在线训练
  硬件平台绑定   通过设备序列号绑定真实机器人（如 S100、MiniDog）
  云端部署       支持将模型一键部署到硬件（SSH / OTA 模式）

------------------------------------------------------------------------

### 4.5 系统后台管理

  功能项       描述
  ------------ ---------------------------------
  用户管理     管理注册用户、套餐、权限升级
  数据集审核   管理上传数据集、合法性检查
  任务调度     查看云端任务状态、重启/停止任务
  计费与日志   查看资源使用统计、账单

## 五、非功能需求

  分类       目标
  ---------- ---------------------------------------------
  安全性     全站 HTTPS，JWT 鉴权，上传文件隔离存储
  性能       API 响应时间 \< 200ms；任务调度延迟 \< 5s
  可扩展性   模块化架构，支持多云部署（Aliyun/AWS）
  可维护性   CI/CD 自动化部署（GitHub Actions + Docker）
  兼容性     支持桌面 Chrome / Edge / Safari 最新版

## 六、系统交互流程（示例）

**示例：Free → Plus 用户完整路径**

    用户注册 → 登录 → 浏览公开数据集 → 申请升级Plus →
    上传私有数据集 → 启动训练任务 →
    查看训练日志 → 推理验证 → 下载模型
