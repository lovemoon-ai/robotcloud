# RobotCloud Frontend

This package contains the web client for RobotCloud, implemented with Next.js 14 and React 18. It provides the operator console described in `design/RobotCloud_FE_Design.md`, including login, dashboard, dataset, training, inference, simulator, and admin modules.

## Prerequisites

- Node.js 18.17 or newer (recommended: the current LTS release)
- npm 9 or newer (bundled with Node.js LTS releases)

Before running the project for the first time, install dependencies:

```bash
cd frontend
npm install
```

If you use Node Version Manager (nvm), you can run `nvm use` in the repository root after selecting an appropriate version.

## Available Scripts

All commands below are executed from the `frontend` directory.

### `npm run dev`

Starts the development server on [http://localhost:3000](http://localhost:3000). The app reloads automatically when you edit files. Environment variables can be defined in a `.env.local` file.

### `npm run build`

Creates an optimized production build under `.next/`. Run this before deploying the application.

### `npm run start`

Runs the production build generated by `npm run build`. By default it listens on port 3000; you can change this via `PORT=<number> npm run start`.

### `npm run lint`

Executes the Next.js ESLint configuration to validate code style and catch common issues.

### `npm test`

Runs the Jest unit tests for the API client. Use the `--watch` flag for interactive mode: `npm test -- --watch`.

## Environment Variables

To integrate with backend services, create a `.env.local` file in the `frontend` directory and define the following variables as needed:

- `NEXT_PUBLIC_API_BASE_URL` – Base URL for backend API requests (defaults to `/api` when omitted)

Add additional environment variables here as backend integrations evolve.

## Additional Notes

- When deploying, ensure the backend API is accessible from the same origin or configure CORS accordingly.
- Run `npm install` again whenever `package.json` or `package-lock.json` changes to sync dependencies.
